Benchmark: [BMIM][NTf₂]

1-Butyl-3-methylimidazolium bis(trifluoromethylsulfonyl)imide is the most widely referenced ionic liquid standard. The NIST ILThermo database contains comprehensive thermophysical data at 298 K and atmospheric pressure, making it an ideal model validation benchmark.

⚗️

Temperature & Pressure

The model takes SMILES + T (K) + P (kPa) as inputs. Conditions must be set manually to match target literature or experimental values. Default: 298.15 K / 101.325 kPa (ambient standard state).

📐

About MIPGraph

MIPGraph encodes cation–anion 3D graphs via a graph neural network and outputs six thermophysical property predictions through multi-task heads. Results are for academic reference; please cite the model and applicable training data range.
